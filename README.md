# BellFall — First Landing Page

BellFall is a simple landing page created as my first web project for the TOP (The Odin Project) curriculum.

## About
This repository contains a small static site demonstrating a landing page layout, hero section, team cards, quote and CTA.

## index.html — page content
The main page (index.html) includes:
- Header with project title and navigation links (Home, About, Contact).
- Hero section with main heading, descriptive paragraph, CTA button and a large hero image.
- Team cards section showcasing four characters with images and short descriptions.
- Quote section with a highlighted italic quote and author attribution.
- Call-to-action (CTA) box prompting users to join the adventure.
- Footer with author credit.


## Technologies
- HTML5 (index.html)
- CSS3 (style.css) — uses the Roboto font in styles
- Static assets in the `images/` folder (PNG files)
